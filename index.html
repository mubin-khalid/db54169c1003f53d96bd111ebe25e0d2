Welcome,

Here you can find the latest updates about the PDOnline Scrapper.

PDOnline Scrapper uses proxies to scrape the site and uses HTML DOM PARSER to manipulate the HTML structure.

Spider is based on the PHP's cURL library, but there is an option for the Yahoo Query Language(YQL) as well. You are welcomed to use that function too(You can find these functions insider Spider.php file). Moreover it uses the CSV Parser to parse the CSV files.
